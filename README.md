# 📌 Overview

AI Resume Builder is a modern full-stack web application that helps users build professional, ATS-friendly resumes with the assistance of **Google Gemini AI**.

Users can create resumes from scratch, enhance content using AI, switch between multiple professionally designed templates, and export resumes as high-quality PDFs while preserving clickable hyperlinks.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/pancholipankaj3008/AI-Powered-Resume-Builder.git
```

Frontend

```bash
cd client
npm install
npm run dev
```

Backend

```bash
cd server
npm install
npm run dev
```
